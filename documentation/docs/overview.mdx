# Build Apps with Reboot

<div style={{ fontSize: "1.2rem" }}>
  Reboot lets you build stateful, multiplayer apps on a durable
  backend. Build AI chat apps that run inside ChatGPT, Claude, and
  VS Code, or full-stack web apps with React frontends.
</div>
