<Step
  title="Generate React API files"
  description={`

Add React code generation to the \`.rbtrc\` file so that the React hooks are generated directly into your frontend project.

`}>

Add the following line to your existing `.rbtrc` file:

```sh
generate --react=web/src/api
```

In the terminal running `rbt dev run` you'll see that the change is
detected and the React code is generated.

</Step>

<Step
  title={<div>Replace exiting code in <code>src/App.tsx</code></div>}
  description={`

Implement a simple React app that displays messages and allows sending new ones using the generated Reboot React hooks.

`}>

```typescript title=src/App.tsx
// ./src/App.tsx
import { useState } from "react";
import { useChatRoom } from "./api/chat_room/v1/chat_room_rbt_react";
// TODO: import "./App.css";

const App = () => {
  const [message, setMessage] = useState("Hello, Reboot!");

  const { useMessages, send } = useChatRoom({ id: "reboot-chat-room" });
  const { response } = useMessages();

  const handleSend = async () => {
    if (message.trim()) {
      await send({ message });
      setMessage("");
    }
  };

  return (
    <div className="app-container">
      <h3 className="app-title">This is Reboot 🎉</h3>

      <div className="messages-container">
        {response?.messages.map((msg, index) => (
          <div key={index} className="message">{msg}</div>
        ))}
        {/* TODO: render pending messages optimistically. */}
      </div>

      <div className="input-container">
        <input
          value={message}
          onChange={(e) => setMessage(e.target.value)}
          onKeyDown={(e) => e.key === "Enter" && handleSend()}
          placeholder="Type your message here..."
          className="message-input"
        />
        <button onClick={handleSend} className="send-button">
          Send
        </button>
      </div>
    </div>
  );
};

export default App;
```

</Step>

<Step
  title="Add optimistic updates (optional)"
  description={`

For a better user experience, you can show pending messages immediately while they're being sent to the backend.

`}>

Replace this line in your `App` component:

```typescript
{/* TODO: render pending messages optimistically. */}
```

With this code:

```typescript
{send.pending.map(({ request: { message } }, index) => (
  <div key={`pending-${index}`} className="pending-message">
    {message} (sending...)
  </div>
))}
```

This will show messages immediately when you click Send, even before the
backend confirms they've been processed. Try it out by making your
backend sleep for a few seconds in the `send` method!
