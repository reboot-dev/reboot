# Queue

A general purpose queue that allows you to enqueue and dequeue [`Item`](/library_services/item)s
singularly or in bulk to it. It is durable, which means that it will persist across
restarts or system failures, and transactional, so you can access it safely.

Each `Queue` will need a unique ID and is created implicitly the first time
you use it. [`Item`](/library_services/item)s can be [`Enqueue`](#enqueue)d as a
single `Value`, `bytes`, or `Any` object or can be [`Enqueued`](#bulk-enqueue)ed
in bulk as a list of `Item`s. They can then be [`Dequeue`](#dequeue)d singularly
in [bulk](#bulk-dequeue) as well. In practice, you will want to use just one of
these formats in a given `Queue`. Read more about [`Item`](/library_services/item)s
to determine the best format for your data.

NOTE: `Any` is only supported in Python, not TypeScript.

    from_bool,
    from_dict,
    from_int,
    from_list,

        Item(value=from_str("apple")),
        Item(value=from_list(["a", "b", "c"])),
