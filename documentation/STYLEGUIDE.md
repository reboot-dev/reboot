# Style Guide

## Follow the Google documentations style guide:
https://developers.google.com/style

## Google style guild reminders:
- Use the second person "you" as opposed to "users".
  E.g. YES: You should never notice you are writing a distributed application.
       NO: Users should never notice they are writing a distributed application.
- DO NOT use 'we' to refer to Reboot.dev, Reboot, or the combination of our users
  and us.
  E.g. YES: You will build a Reboot app today.
       NO: We will build a Reboot app today
       YES: Use the Reboot 'Getting  Started' Helm chart.
       NO: Use our Helm chart to get started.

## Reboot.dev and Reboot specific styles.
Backtics for `readers`, `writers` and `transactions`. With links [`reader`](https://gooogle.com) , if necessary.

Node.js uses this spelling "Node.js" everywhere.

"Python" is always capitalized.
